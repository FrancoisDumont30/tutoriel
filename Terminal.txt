-------
Dossier
-------

XXX -- help	-> show XXX help 
man fonction	-> show help
q		-> exit


Pwd 		-> montre le dossier actuel
ls		-> montrer les fichiers et dossiers dans le dossier actuel
ls – la		-> montrer sous forme de liste (a : pour fichier caché)


cd nom 		-> accéder au dossier « nom »
cd ..		-> revenir en arrière
cd ../..	-> revenir en arrière 2x
cd ~		-> revenir au dossier parent


mkdir "nom"	-> créer le dossier nom
mkdir ../nom	-> créer le dossier « nom »

rm nom		-> supprimer le fichier
rm -r nom	-> supprimer le dossier nom

touch nom.py	-> créer un fichier 


mv file_name folder_name	
		-> move the file "file_name" in the forlder "folder_name"
		
cp file_name folder_name	
		-> paste the file "file_name" in the forlder "folder_name"

cp file_name new-file_name

cp -r folder/ copy_name 
		-> paste the folder with new name.

Shortcuts
---------

Molette de souris 	-> coller la séléection
Arrows Up / Down	-> naviguer dans l'historique des commandes
tab			-> autocompletion
Ctrl + r		-> recherche dans l'historique des commandes
Ctrl + a		-> aller au début de la ligne actuelle
Ctrl + e		-> aller à la fin de la ligne actuelle
*			-> select everything
			move * folder_name : move evry file to the folder
			move

grep XXX file.xyz	-> look for XXX in the file.xyz
